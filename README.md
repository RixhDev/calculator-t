# Calculator in C

Calculator developed in C language optimized to run in terminal, Powershell, Windows console, Termux and make basic operation's (add, rest, multiply, division about enter numbers).

The project is designed to be **portable**, **modular** and easy to compile on Linux/Unix systems, Powershell and Windows console.

---

## Features

- Interactive terminal menu.
- Input validation (prevents errors due to invalid data).
- Division by zero handling using `NAN`.
- Modularity: functions separated into header (`include/`) and source (`src/`).
- Automated compilation with `Makefile`.

---
## How to use (General)

### *1. Clone the repo*

	git clone https://github.com/RixhDev/calculator-t

### *2. Go to the proyect folder*

	$ cd Calculadora

### *3. Look the proyect structure*

	$ ls

	You should see:
	src/  include/  Makefile  README.md

### *4. Compile*


	$ make
#### *if you are run in Termux go to Termux installation...

### *5. Run*

	$ ./calculadora-t

### *6. [OPTIONAL] If you want to clean*

	$ make clean
	*This command are delete the file calculadora-t generated by make.

## Termux:

The original Makefile file uses *gcc* for compilation; you need to change it to *clang*.
For this... Open the Makefile with nano/vim in your Termux, after Change the first line for this 

### (Prerequisite) Install dependencies
You must have:

	pkg update && pkg upgrade 
	pkg install make clang
	
Line 1 before:
	
	CC= gcc

Line 1 now:
	
	CC= clang

✅ That's it, now you can run Make successfully (step 5).

## Project Structure

Standar function of I/O: 'include <stdio.h>'
Aux function: 'include <stdlib.h>'
Math function and constants: 'include <math.h>'

mostrarMenu(void): Muestra el menú principal con las operaciones disponibles.

limpiarBuffer(void): Esta función descarta caracteres residuales en el buffer de entrada, evitando que lecturas posteriores fallen por entradas inválidas.

solicitarCantidad(int min, int max): Solicita al usuario la cantidad de números a operar, validando que esté dentro de un rango permitido.

solicitarNumero(const char mensaje): Solicita un número desde teclado, repitiendo hasta que la entrada sea válida.

operar(int opcion, int n, float nums[]): Ejecuta la operación matemática seleccionada (suma, resta, multiplicación o división).

mostrarMenu(void): Centraliza la presentación de opciones al usuario, facilitando cambios futuros.

División: divide secuencialmente, en caso de una division por cero se ha asignado el retorno de NAN.

main(void): Controla el flujo general

	1) Muestra el menú.

	2) Valida la opción elegida.

	3) Solicita cantidad y números.

	4) Ejecuta la operación.

	5) Muestra el resultado o error si es necesario.
